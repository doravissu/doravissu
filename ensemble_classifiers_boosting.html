In [ ]:
For tree 10, what is the misclassification rate
In [ ]:
E = (0.5*1+0.2*0+)/0.5+0.2+0.1+0.04 = 0.65
In [ ]:
Alpha = learning rate * log((1-E)/E)
In [ ]:
Error prone model has lower alpha, and accurate model has low E and high alpha
In [ ]:
Updating observation weights = Wt *(-alpha * actual * predicted)
In [ ]:
For incorrectly classified observation:
W11 (7) = Wt10*e(-0.42*1*-1)
W11 = Wt10*e(0.42) = Wt10*2
In [ ]:
For correctly classified observation:
W11(8) = Wt10*e(-0.42*1*1)
W11 = Wt10*e(-0.42) = Wt10*0.7
